这是位于正常位置的标题

相对定位会按照元素的原始位置对该元素进行移动。

这个标题相对于其正常位置向左移动，样式 "left:-20px" 从元素的原始左侧位置减去 20 像素。

这个标题相对于其正常位置向右移动，样式 "left:20px" 向元素的原始左侧位置增加 20 像素。

我是box2的父元素
box2采用绝对位置，处于left:100px, top= 300px(生成绝对定位的元素，相对于 static 定位以外的第一个父元素box1进行定位)
我是box3采用固定定位,相对于浏览器窗口进行定位
static 默认值。没有定位，元素出现在正常的流中（忽略 top, bottom, left, right 或者 z-index 声明）。
我在哪,绝对定位的盒子脱离了标准文档流
